## Contents

- `index.html` — the complete single-page site (self-contained HTML, CSS, and JS).

## Run locally

Open `index.html` in any web browser. No build step or dependencies required.

## Deploy with GitHub Pages

1. Go to **Settings → Pages**.
2. Under **Build and deployment → Source**, choose **Deploy from a branch**.
3. Select the **main** branch and the **/ (root)** folder, then **Save**.
4. The site publishes at `https://pandeydumka.github.io/ismassociates/`.
